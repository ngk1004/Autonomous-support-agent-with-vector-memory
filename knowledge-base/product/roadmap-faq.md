# Product Roadmap FAQ

Answers to common questions about AcmeCloud's product direction and release schedule.

## Where can I see the roadmap?

- **Public roadmap**: https://roadmap.acmecloud.example (high-level, no dates)
- **Changelog**: https://acmecloud.example/changelog (shipped features)
- **Enterprise customers**: Quarterly roadmap briefings with your CSM

We do not publish specific delivery dates for unshipped features to maintain flexibility.

## Release Cadence

- **Minor features**: Bi-weekly releases (Tuesdays)
- **Major features**: Quarterly
- **Security patches**: As needed, within 24–72 hours for critical issues

## Recently Shipped (Last Quarter)

- Webhook event replay (Enterprise)
- API usage dashboard
- SAML SSO for Pro plans
- Improved deployment rollback UI

## Upcoming (No Guaranteed Dates)

- Slack integration
- Microsoft Teams notifications
- Advanced audit logs for Pro
- Self-hosted deployment option (Enterprise beta)

## Beta Program

Join **Settings → Labs** to opt into beta features. Beta features may change or be removed without notice.

## Deprecation Policy

- Deprecated features announced 90 days in advance
- Email notification to admins
- Migration guides published in docs
- Breaking API changes versioned (`/v2/` endpoints)

## Influencing the Roadmap

The best way to influence our roadmap:

1. Submit detailed feature requests
2. Upvote on the feedback portal
3. Participate in annual customer advisory board (Enterprise invite-only)
